# Offline Data Collection Application

This project is an **offline-first mobile survey platform** designed for data collection in low-connectivity and remote environments. The mobile app allows field workers to create and fill surveys without internet access and automatically sync responses to the server once connectivity is restored.

It supports **skip logic, conditional question flows, and input validation** to ensure structured and reliable data collection. A companion web dashboard enables organizations to design survey forms, manage collected responses, and view basic analytics, making it suitable for NGOs and development teams conducting large-scale field surveys.

## Key Features
- Offline survey creation and response storage 📱
= Automatic background sync on reconnection 🔄
- Skip logic and conditional workflows
- Data validation for improved quality
- Web interface for form management and analytics 📊
